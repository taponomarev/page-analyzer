## Requirements

* PHP ^8.0
* Extensions: mbstring, curl, dom, xml,zip, sqlite3
* Composer
* Node.js & npm
* SQLite for local
* [heroku cli](https://devcenter.heroku.com/articles/heroku-cli#download-and-install)

## Setup

```sh
$ make setup
```

## Run

```sh
$ make start
```
